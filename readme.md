# Select Your XI 🏏

Select Your XI is a simple React application where users can build their own cricket playing XI by selecting players from an available player list. When a player is selected, they move to the selected section and their price is added to the total budget, similar to an add-to-cart system.

## 🚀 Features

- Display player cards in the **Available Players** section
- Select players to add them to **Selected Players**
- Player moves from available list to selected list
- Player **price is added to total cost**
- Simple and responsive UI
- Built using modern frontend tools

## 🛠️ Technologies Used

- React
- Tailwind CSS
- DaisyUI
- JavaScript (ES6)

🎯 Project Goal

The main goal of this project is to practice React component structure, state management, and UI building while creating a simple interactive cricket team selection system.
